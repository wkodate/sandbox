# サーバレスとはなにか

開発者が本当にやりたいことは、ユーザに価値を届ける。これらをマネージメントサービスに任せることができる

# ハンズオン

* 流れ
    * AWS Lambdaで日→英翻訳する
    * 翻訳Web APIを作る
    * 文字起こし+翻訳パイプラインを作る

* EC2と比較して自由度は低いが、その分手間も少ない
* サーバのプロビジョニング/管理無しでプログラムを実行できるサービス
* 料金体系はリクエストベース
* 作業の流れ
    * Lambda Functionを作成し、Hello Worldと表示する
    * Lambda Functionを修正し、Amazon Translateと連携する
